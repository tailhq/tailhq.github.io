StochasticRadialDiffusion 

Companion object StochasticRadialDiffusion

class StochasticRadialDiffusion[ParamsQ, ParamsD, ParamsL] extends Serializable

A radial diffusion system where the injection and diffusion functions are treated as latent i.e. not directly observed.

The processes of injection Q(l,t) and diffusion DLL(l,t) are modeled using the io.github.mandar2812.dynaml.models.bayes.CoRegGPPrior implementations from DynaML.

df/dt = L2d/dL(DLL × L-2 × df/dL) + Q(L,t)

Q(L,t) ~ GP(q(L,t), Cql(L,L'), Cqt(t,t'))

DLL(L,t) ~ GP(d(L,t), Cdl(L,L'), Cdt(t,t'))

See RadialDiffusion for an introduction to the radial diffusion solver.
