class MagRadialDiffusion[T] extends Serializable

Implementation of a discrete radial diffusion system, parameterised by magnetospheric processes.

∂ƒ/∂t = L2∂/∂L(DLL × L-2 × ∂ƒ/∂L) - λ(L,t) × ƒ(L,t) + Q(L,t)

This class solves the radial diffusion dynamics, when supplied with the magnetospheric parameters.
