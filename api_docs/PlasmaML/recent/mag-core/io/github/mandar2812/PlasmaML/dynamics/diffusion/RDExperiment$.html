    Generate synthetic PSD data for an inference experiment.

    Generate synthetic PSD data for an inference experiment.

    rds

    A radial diffusion solver of type RadialDiffusion

    dll

    Diffusion field.

    lambda

    Loss rate.

    q

    Particle injection rate.

    initialPSD

    The PSD at t = 0

    noise

    Measurement noise to be added to the generated PSD profile.

    num_boundary_points

    Number of points from the initial PSD state to be sampled.

    num_bulk_points

    Number of points to be sampled from the interior of the space-time domain.

    num_colocation_points

    Number of "co-location" points, i.e. points where the radial diffusion PDE is enforced (with some slack).

    returns

    The generated PSD profile, noisy observations (boundary and bulk) and co-location points.
