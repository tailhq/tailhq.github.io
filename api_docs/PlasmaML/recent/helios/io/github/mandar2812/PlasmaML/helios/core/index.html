  2.  case class CausalDynamicTimeLag[P, T, L](name: String, size_causal_window: Int, prior_wt: Double = 1.5, error_wt: Double = 1.0, temperature: Double = 1.0, specificity: Double = 1.0, divergence: Divergence = ..., target_distribution: TargetDistribution = CausalDynamicTimeLag.Boltzmann)(implicit evidence$1: TF[P], evidence$2: IsFloatOrDouble[P], evidence$3: TF[T], evidence$4: IsNumeric[T], evidence$5: TF[L], evidence$6: IsFloatOrDouble[L]) extends Loss[((Output[P], Output[P]), Output[T]), L] with Product with Serializable

    The loss functions assumes the prediction architecture returns 2 × h number of outputs, where h = size_causal_window.

    Causal Dynamic time-lag Inference: Weighted Loss

    The loss functions assumes the prediction architecture returns 2 × h number of outputs, where h = size_causal_window.

    The first h outputs are predictions of the target signal, for each time step in the horizon.

    The next h outputs are unscaled probability predictions regarding the causal time lag between the input and output signals, computed for any input pattern..

    The loss term for each data sample in the mini-batch is computed independently as follows.

    L = Σ i (|fi - yi|2 × (1 + c × pi)) + γ D(p*, p)

    name

    A string identifier for the loss instance.

    size_causal_window

    The size of the finite time horizon to look for causal effects.

    prior_wt

    The multiplicative constant applied on the "prior" term; i.e. a divergence term between the predicted probability distribution of the time lags and the so called "target probability distribution".

    temperature

    The Gibbs temperature which scales the softmax probability transformation while computing the target probability distribution.

    specificity

    Refers to the weight c in loss expression, a higher value of c encourages the model to focus more on the learning input-output relationships for the time steps which it perceives as more probable to contain the causal time lag.

    divergence

    The kind of divergence term to be used as a prior over the probability distribution predicted for the time lag. Available options include. - Hellinger distance - Kullback-Leibler Divergence (default) - Cross Entropy - Jensen-Shannon Divergence

    A weighted loss function which enables fuzzy learning of the solar wind propagation from heliospheric images to ACE.

    RBF-Kernel Weighted Solar Wind Loss (KSW Loss)

    A weighted loss function which enables fuzzy learning of the solar wind propagation from heliospheric images to ACE.

    name

    Unique string identifier of the loss/layer

    size_causal_window

    The size of the finite time horizon, over which time-lag and signal predictions are made.

    kernel_time_scale

    The time scale of the Radial Basis Function (RBF) convolution kernel, defaults to 1.

    kernel_norm_exponent

    The Lp norm used in the kernel, defaults to 2

    corr_cutoff

    The cut-off applied to the Spearman correlation between the predicted targets and time lags, defaults to -0.75.

    prior_scaling

    A scaling parameter multiplier applied in the prior term, defaults to 1.

    batch

    The batch size for each training epoch.
