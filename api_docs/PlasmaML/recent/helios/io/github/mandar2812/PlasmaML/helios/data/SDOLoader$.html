  7.  def download(path: Path, createDirTree: Boolean = true)(instrument: String, size: Int = 512)(date: LocalDate): Unit

    Download images taken on a specified date.

    Download images taken on a specified date.

    path

    The root path where the data will be downloaded, the downloader appends soho/[instrument]/[year] to the path supplied and places the images in there if the createDirTree flag is set to true.

    createDirTree

    If this is set to false, then the images are placed directly in the path supplied.

    instrument

    The instrument code as a string, see SDOData.Instruments

    size

    The resolution of the images, defaults to 512 × 512

    date

    a Joda time LocalDate instance.
