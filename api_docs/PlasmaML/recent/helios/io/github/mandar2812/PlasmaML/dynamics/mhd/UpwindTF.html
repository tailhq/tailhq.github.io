Upwind Solar Wind Model

1-Dimensional up-wind solar wind propagation model, implemented as a tensorflow computational layer.

rDomain

The lower and upper limits of the radial domain.

nR

Number of divisions of the radial domain.

omegaInit

Initialiser for the solar rotation speed.
