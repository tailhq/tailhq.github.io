    Create a DynaML data collection for the supervised solar wind prediction task.

    Data Generation: FTE - Solar Wind Prediction

    Create a DynaML data collection for the supervised solar wind prediction task. The data collection consists of:

    • Inputs: A vector consisting of FTE and magnetic field values for a specific time stamp
    • Outputs: A time series of OMNI solar wind values; which may or may not be processed.
    fte_data_path

    The file system path containing FTE and magnetic field files (FTE assumed to be text, while BRCP/BRSS are in FITS format).

    c

    A configuration object (see FteOmniConfig) containing information on how to construct the input and output patterns.

    ts_transform_output

    A data pipeline which processes each output time series pattern, if not relevant, simply provide the identity mapping.

    tt_partition

    A data pipeline which decides which data patterns lie in training or test data sets.

    exogenous_inputs

    A set of OMNI data set quantities which should be used as exogenous data sources.
