    Subroutine to generate synthetic input-lagged output time series.

    Subroutine to generate synthetic input-lagged output time series.

    x(n+1) = (1 - α). R(n) × x(n) + ε

    y(t + Δt(x(t))) = f[x(t)]

    Δt(x(t)) = g[x(t)]

    compute_output_and_lag

    A data pipe which takes the input x(t) a tensor and computes y(t + Δt(x(t))) the output and Δ(x(t)), the causal time lag.

    sliding_window

    The size of the sliding time window [y(t), ..., y(t+h)] to construct. This is used as training label for the model.

    d

    The dimensions in the input time series x(t)

    n

    The length of the time series x(t)

    noiserot

    The variance of elements of Ri,j, a randomly generated matrix used to compute an orthogonal transformation (rotation) of x(t).

    alpha

    A relaxation parameter which controls the auto-correlation time scale of x(t)

    noise

    The variance of ε
