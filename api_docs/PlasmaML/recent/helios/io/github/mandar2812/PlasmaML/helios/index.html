    Run a CDT experiment, predicting solar wind arrival time and speed using SOHO/SDO images.

    Run a CDT experiment, predicting solar wind arrival time and speed using SOHO/SDO images.

    dataset

    A collection of PATTERN instances i.e. date time stamps, image paths and upstream solar wind values. See HELIOS_OMNI_DATA.

    tt_partition

    A function which takes each PATTERN and returns true if it should be kept in the training set and false otherwise.

    architecture

    Neural network architecture. Takes a tensor representation of an image (or time series of images) and predicts.

    • Running history of solar wind speed
    • Probability distribution for causal time lag link
    hyper_params

    A list of hyper-parameters.

    loss_func_generator

    A function which takes some value of hyper-parameters, and generates the CDT loss function. See CausalDynamicTimeLag for more information on its hyper-parameters.

    fitness_func

    A data pipe which takes the model predictions, and the ground truth solar wind time series and computes a fitness score which is used to compare hyper-parameter assignments.

    hyper_prior

    A prior distribution over the hyper-parameter space, the log likelihood computed from this is added to the fitness to yield the total score for a hyper-parameter configuration.

    results_id

    A string identifier for the results directory, this folder will contain each trained model including the resulting model after the tuning procedure.

    resample

    Set to true if the training data should be resampled to mitigate imbalance of low versus high solar wind episodes.

    preprocess_image

    A data pipe which applies some user specified pre-processing operations on each image.

    image_to_bytearr

    A data pipe which converts the image to an array of bytes.

    processed_image_size

    Size in pixels of the processed image used to construct tensor patterns.

    num_channels_image

    Number of colour channels in each processed image pattern.

    image_history

    If set to a positive value, a stacked time history of images is constructed for each data pattern.

    image_history_downsampling

    Sets the down-sampling frequency of the image histories.

    iterations

    Maximum number of iterations to train the selected model during final training.

    iterations_tuning

    Maximum number of iterations to train model instances during the tuning process.

    summaries_top_dir

    The top level directiry under which the results directory of the experiment is to be created.

    optimizer

    Optimization algorithm to be used in tuning and training.

    miniBatchSize

    The mini batch size.

    num_hyp_samples

    The number of hyper-parameter samples (population size) to use in Grid Search gs and Coupled Simulated Annealing csa subroutines.

    hyper_optimizer

    Hyper-parameter search technique.

    • gs: Grid Search (in this case random search). The default value
    • csa: Coupled Simulated Annealing
    • cma: Covariance Matrix Adaptation - Evolutionary Search
    hyp_opt_iterations

    In case of csa and cma methods, the maximum number of iterations to execute.

    hyp_mapping

    A one-to-one functional mapping for each hyper-parameter from its original domain to the Real number line. Optional and defaults to None.

    returns

    An Experiment instance which contains a ModelRunTuning instance and some configuration information ImageExpConfig.
