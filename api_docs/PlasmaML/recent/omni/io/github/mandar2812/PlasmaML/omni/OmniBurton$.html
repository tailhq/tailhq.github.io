OmniBurton 

object OmniBurton

Contains work-flows for running experiments on hybrid GP-Burton injection inference and prediction models.

The basic premise of this code is the Burton ODE for time evolution of the Dst index

d Dst(t)/dt + lambda*Dst(t) = Q(t)

Q(t) ~ GP(0, K(t, t'))
