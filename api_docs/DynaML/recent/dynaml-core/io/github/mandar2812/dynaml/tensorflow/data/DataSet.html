class DataSet[X] extends AnyRef

DynaML Data Set

The data set class, wraps an underlying collection of elements of type X.

It can be used to create an object which can access a potentially large number of data patterns.

It is also possible to transform the data collection in the classical functional paradigm of map(), flatMap().

X

The type of each data pattern.
